Interopérabilité

Définition

L’interopérabilité en SIG (Système d’Information Géographique) désigne la capacité de différents systèmes, logiciels ou bases de données géographiques à échanger, comprendre et utiliser des informations géospatiales de manière fluide et cohérente. Autrement dit, c’est la capacité d’un SIG à fonctionner avec d’autres SIG ou applications sans perte d’information.

Enjeux

  • Faciliter l’échange d’informations

  • formats de fichiers, protocoles et standards.

  • signification cohérente des données pour tous les utilisateurs.

  • Respect des directives telles que la directive européenne INSPIRE pour la diffusion des données géographiques.

Normes et standards

Open Geospatial Consortium (OGC)

Organisation internationale qui définit des standards ouverts pour les données géospatiales.

Exemples :

  • WMS (Web Map Service) : service de cartes sur le web

  • WFS (Web Feature Service) : service d’accès aux entités géographiques

  • GML (Geography Markup Language) : langage XML pour décrire les données géographiques

Geo-standards

Normes ISO pour les données géospatiales (ex. ISO 19115 pour la métadonnée).

Assurent la compatibilité entre systèmes et la qualité des échanges.

Résumé : ESRI c'est pas trop interopérable :)
